    插值¶

    1. 插值三问:
    2. 插值是为什么? 好看(即答) 连续性
    3. 用什么插值? Polynomial
    4. 经典之拉格朗日插值. 没啥好说的: 可证明唯一性; 误差分析 应用: 分享秘密: 6个人合体:还原出目标多项式; 5个人=0个人 龙格现象: 边界的剧烈抖动
    5. 切比雪夫插值多项式: 解决龙格现象
    6. 动机: 提高对插值误差的最大值的控制: \frac{\prod_{i=1}^n(x-x_i)}{n!}f^{(n)}(c)
    7. 假设 f: [-1, 1] \rightarrow \mathbb{R} \
    8. 目标: 选取 x_1, x_2, \cdots, x_n 最小化 || \prod(x-x_i) ||_{\infty}
    9. x_i = \cos{\frac{(2i-1)\pi}{2n} }
    10. Chebyshev插值多项式 T_n(x):=2^{n-1}(x-x_1)(x-x_2)\cdots(x-x_n)
    11. 等价于 T_n(x):= \cos{(n \arccos x)}
    12. 递归关系: T_{n+1}(x) = 2xT_n(x) - T_{n-1}(x)
    13. 可以证明的是, 这是令误差最小的插值方式

    逼近¶

    1. Weierstrass定理: 连续函数可以任意地被多项式逼近
    2. Bernstein Polynomial:
    3. 给定: f: [0,1] \rightarrow R
    4. 目标: 构造多项式 p, 使得在 x \in [0,1] 这一点上, p(x) 尽可能接近 f(x).
    5. K \sim B(n, x)
    6. E[k]=nx, Var[K]=nx(1-x), Pr[K=i] = {n\choose i} x^i(1-x)^{n-i}
    7. Bernstein Poly的一组基: b_{n,i}= {n\choose i} x^i(1-x)^{n-i}
    8. \displaystyle \lim_{n\rightarrow \infty}Pr[|\frac K n - x| > \delta] 对所有 x 一致成立
    9. Bernstein Poly的本体: E[f(\frac K n)] = \sum_{i=0} f(\frac i n)b_{n, i}(x)
      • 个人理解: 等距采样, 加权平均(权是一个分布多项式函数)
    10. 证明: 拆成两部分

    范数¶

    1. 一般向量范数: \displaystyle ||x||_k = (\sum{|x_i|^k})^{\frac1 k}
    2. 函数范数: ||f||_{\infty} = \sup |f|, ||f||_2 = (\int |f(x)|^k dx)^{\frac 1 k}

    最小二乘法¶

    1. 几何: 简单投影 A^TA\overline{x} = A^Tb
    2. 微积分: 最小化 (Ax-b)^T(Ax-b), 求解梯度 \Delta = 2A^TAx-2A^Tb = 0.
    3. Gram-Schmidt: 消除已经存在的维度上分量.

    狐里叶变换¶

    1. DFT FFT IFFT
    2. 动机: 多项式乘法: p(x) * q(x)
    3. 给定 p(x), q(x) 的系数
    4. 令 m \geqslant 2n+1, 选定系数 x_0, x_1, \cdots, x_m, 计算 $p,q $ 在上面的点值.
    5. 计算 r(x_i) = p(x_i)q(x_i)
    6. 对 r 插值, 还原出 r 的系数
    7. 愿景: 2, 4 都能做到 O(n\log n)
    8. DFT:
    9. 选取 x_i 为 m+1 次单位根.
    10. 逆变换: 插值
    11. 但是, 还是很慢,,,
    12. FFT:
    13. 引入分治, 将 p(x) 分成奇数和偶数:
      • E(z) = a_0 + a_2z + \cdots
      • O(z) = a_1 + a_3z + \cdots
      • p(z) = E(z^2) + zO(z^2)
    14. IFFT:
      • DFT, FFT: p(\omega^l)= \frac1{\sqrt{n+1}}\sum a_j \omega_{lj}
      • IFFT: a_l = \sum p(\omega^j) \omega^{-lj}, 和DFT唯一区别是 \omega \rightarrow \omega^{-1}

    解方程组之迭代Method¶

    1. 线性方程组的数值稳定性: Ax = b
    2. 定义条件数 cond(A) = \frac{x相对误差}{b相对误差}
    3. cond(A) = \max{\frac{||A^{-1}e||/||A^{-1}b||}{||e|| / ||b||}} = \max{\frac{||A^{-1}e||}{||e||}} \max{\frac{||Ax||}{||x||}}
    4. 引入矩阵的算子范数 ||A||:= \max{\frac{||Ax||}{||x||}}
      • 特别地有: ||A||_2:= \max{\frac{||Ax||_2}{||x||_2}} = \sqrt{\lambda_{max}(A^TA)}
    5. cond(A) = ||A|| \cdot ||A^{-1}||
    6. 高斯消元法: 朴素的 O(n^3) 做法
    7. Jacobi迭代:
    8. 收敛的充分条件: 严格对角占优(对角线上元素超过行内其他元素)
    9. 不动点迭代形式: A=L+D+U, Ax=b 可以写为 x=D^{-1}(b-(L+U)x)
    10. Gauss-Seidel: (L+D)x=b-Ux
    11. 收敛性: x_{k+1} = Ax_{k} + b
      • 令 x_* 为一个不动点
      • 有 x_k-x_{ * } = A^k(x_0-x_{ * })
      • 收敛条件:
      • A^k \rightarrow 0
      • x_0 - x_* 存在于一个好的线性子空间里.
      • 谱半径: 最大的{特征值的绝对值}
    12. Jacobi迭代的谱半径: \rho(D^{-1}(L+U)), 若满足严格对角占有, 可以证明\rho < 1
    13. Moore-Penrose伪逆 \overline x = (A^TA)^{-1}A^Tb, (A^TA)^{-1}A^T 是维尼.(前提: 列线性无关)
    14. Richardson迭代:
    15. 目的: A^{-1} = p(A) 的近似.
    16. 等价地, q(x) = 1-xp(x), 需要寻找 q(0) = 1, q(x) \approx 0, \forall x > 0
    17. 要解 Ax=b, 即找出 x_* = p(A)b \in span(b, Ab, A^2b, \cdots)
    18. 迭代形式: x_{k+1} = (I -\alpha A)x_k + \alpha b
    19. 误差 e_k = x_k - x_* 满足 e^k = (I-\alpha A)e_{k-1} = (I-\alpha A)^k e_0
    20. 收敛性: 当且仅当 \rho(I-\alpha A) = |1-\alpha\lambda_{max}| < 1
    21. 这里的 x_k 处于 span(b, Ab, A^2b, \cdots, A^{k-1}b) 中(Krylov子空间)
    22. Chebyshev迭代: 每一轮选取不同的 \alpha, 最小化 ||\prod_i (I-\alpha_i A)||, 使用了Chebyshev多项式.
    23. 内积: 给定正定矩阵 A, 定义内积 $ { < x, y > }_A := x^TAy $.
      • 内积为0, 则称 x,y 关于 A 共轭.
      • 共轭向量一定线性无关.
    24. 考虑不断增大的Krylov子空间序列, 设 x_i 为第 i 个子空间中最逼近 x_* 的元素.
    25. 引理 v_i := x_i - x_{i-1}, 则 {v_i} 两两共轭. (注意到Ax_j - b 必然于 K_j 正交, 因此 Av_j = A(x_j - x_{j-1}) \in K_{j-1}^{不会打的符号}, 而 v_i \in K_i \in K_{j-1}, 因此这两个向量共轭)
    26. 推论 K_i = span(v_1, v_2, \cdots, v_i)
    27. 幂迭代(计算特征值)
    28. 考虑任意向量的特征值分解, 可以找出最大特征值和对应的占优向量,
    29. 迭代可能造成浮点数溢出, 一般每次迭代选择进行归一化.
    30. 迭代得到了一个近似的特征向量方向, 求解最小二乘法 ||Ax-\lambda x||_2^2, 法线方程为 \lambda = \frac{x^TAx}{x^Tx}, Rayleigh Quotient.
    31. 定理, 如果上述目标函数误差小于 \epsilon, 则特征值求解误差同样小于 \epsilon.

    随机游走 Random Walk¶

    1. 定义: 从一个点出发, 每一次都随机走到一个邻居上, 不断重复.
    2. 长期表现会怎样呢?
    3. 重复t步后, 当前顶点是某个u的概率是多少?
    4. 是否存在稳态分布?
    5. 多久才会收敛? (mixing time)
    6. 从s出发多久才到t? (hitting time)
    7. 多久才能遍历每个节点各一次? (cover time) 1,2,3都可以通过纯概率的方法去解决, 4,5见电阻网络.
    8. 马尔可夫链: past is in the past 一切之和当前状态有关.
    9. 转移矩阵描述: \vec{p}_{t+1} = P\cdot \vec{p}_{t}
    10. 对于有限Markov Chain, 若对应的有向图是强连通的, 那说明它是不可约的.
    11. 如果Markov Chain中每个状态都是非周期的, 那么这个Chain是非周期的.
    12. 引理: 对于不可约, 非周期Markov Chain, 一定存在足够大的常数, P^t_{i,j} > 0, \forall t \geqslant T
    13. 稳态: \vec{\pi} = \vec{\pi}\cdot P.
    14. 回归时间: 从 i 出发, 第一次返回到 i 的时间 H_i. h_i = E[H_i]
    15. Markov基本定理: 有限, 非周期, 不可约的Markov链, 必然收敛到一个唯一的稳态分布, \pi(i) = \frac 1 h_i. (怎么证,,,,)
    16. 例子: PageRank
    17. Markov相关内容详细证明见本书
    18. 无向图上的随机游走
    19. $p_{t+1}(v) = \sum_{u:uv\in E}p_t(u)\cdot \frac1{deg(u)} $
    20. p_{t+1} = (AD^{-1}p_t), p_t = (AD^{-1})^t p_0
    21. 稳态分布: \vec{\pi} = \frac{\vec{d}}{2m}, 是 AD^{-1} 的一个右特征向量, 特征值为1.
    22. 无向图上的马尔可夫基本定理: 对于任意有限的, 连通的, 非二分图, 不管从什么养的 p_0 开始, 总有 \vec{p_t} \rightarrow \vec{\pi}.
    23. 惰性随机游走: p_t = \frac12(I + AD^{-1})^t p_0 (对二分图有效)
    24. W 与 D^{-\frac12}WD^{\frac12} = D^{-\frac12}AD^{-\frac12} = \mathcal{A} 相似, 拥有相同的特征值.、
      1. 对于d正则图比较特殊, W = \frac 1 d A= I- \frac 1 d L, 可以将 p_0 展开, 从大到小排序的特征值\alpha_1 = 1, 特征值位于 [-1, 1] 中, a_n = -1 当且仅当图是二分图.
      2. 对联通的非二分图, p_t \rightarrow c_1v_1 = \frac 1 n \vec{1} = \vec{\pi}
      3. 对惰性的随机游走, 特征值发生了 [-1,1] \rightarrow [0,1] 的保持相对距离的映射.
    25. 混合时间:
      • 定义谱间隔 \lambda = \min{1-\alpha_2, 1-|\alpha_n|}
      • 定理: \epsilon- 混合时间.

    图与代数¶

    1. 二分图邻接矩阵: A 特征值关于 x=0 对称.
    2. 图邻接矩阵最大特征值 deg_{avg}(G) \leqslant \lambda_{max}\leqslant deg_{max}(G)
    3. Laplacian Matrix 的一个特征值是 0, 对应特征向量是 \vec{1}, 是半正定的.
    4. 当且仅当特征值 0 的重数为1时, G 连通. x^TLx = \sum(x_i-x_j)^2
    5. Perron-Frobenius定理: 对于非负, 不可约, 非周期的矩阵
    6. 最大特征值重数为1
    7. 对应特征向量中, 每个维度都是非0且同号的.
    8. |\lambda_i| < \lambda_1

    电阻与电路网络¶

    1. 为什么我会在这复习物理QAQ 解决之前的留下的 Hitting&Cover Time 问题.
    2. 基本概念
    3. 基尔霍夫定律: in=out
    4. 欧姆定律: \Delta \phi = ir
    5. 电阻 r_e, 电导率 w_e = \frac1{r_e}
    6. b_v 表示外界流到 v 的电流
    7. 若 w_{uv} = 1(电阻为1), 求解\vec{b} = L \vec{\phi} (That's All We Want)
    8. L = BWB^T, b= L\phi = BWB^T\phi = B\vec{i}
    9. 若存在满足条件的解, 则 b = L\phi \perp \vec{1} (外部流出和流入的相等) (其实这是个充要条件)
    10. L^{+} = \sum_{i=2}^n \frac1{\lambda_i}v_iv_i^T 是维尼. 全体解集是 \{L^+\vec{b} + c \vec{1} \}, 若固定一点电势, 有唯一解.
    11. R_{eff} 等效电阻为电势差. R_{eff}(s,t) = b^T_{st}L^+b_{st}, b_{st}(s) = 1, (t=-1)
    12. Thompson's Principle: R_{eff}(s,t) \leqslant \epsilon(\vec{g}), 单位电流最小化能量.
    13. Commute Time: C_{s,t} = 2mR_{eff}(s,t), m = |E(G)|
    14. 推论: C_{u,v} \leqslant 2m
    15. 连通图遍历时间最多为 2m(n-1)
